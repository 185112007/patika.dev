Kisayollar:
  • Ctrl + P = açılan menuden projenizin dosyasını acabilirsiniz
  • Ctrl + W = cık olan dosyayı kapatmak
  • Ctrl + B = sag tarafta acik olan menuyu kapatmak
  • Ctrl + Shift + P = ayarlara ulasabilirsiniz
  • Ctrl + Shift + Sol Yön Tüşü = bir ifadenin sonun bir ifadenin tamamını secmek
  • Alt + Yukarı/Aşagı Yon Tusu = bir ifadenin yerini asagı veya yukarı yerini degistirmek
  • Alt + Shift = secili satiri bir alt satira kopyalamak
  • Ctrl + Enter = bir alt satira gecmek
  • Ctrl + D = dosyada yer alan aynı ifadeleri secmek
  • Alt + Sol/Sag = ifadeleri teker teker ilerlemek
  • Ctrl + C = secili ifadeyi kopyalamak
  • Ctrl + V = kopyalanan ifadeyi yapıstırmak
  • Ctrl + X = secili ifadeyi kesmek
